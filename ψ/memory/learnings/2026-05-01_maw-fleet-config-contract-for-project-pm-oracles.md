# Maw fleet config contract for Project PM Oracles: each needs project_path (absol

Maw fleet config contract for Project PM Oracles: each needs project_path (absolute path to project repo, not oracle repo), icon (emoji), and short_name in ~/.config/maw/fleet/<slug>.json. File naming: no numeric prefix, just <project-slug>.json. Specialist Oracles (codec, chrome, neon) don't need project_path — they have no fixed project.
