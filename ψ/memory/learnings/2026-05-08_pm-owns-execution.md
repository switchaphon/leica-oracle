# PM Owns Execution, Not Just Coordination

The instinct to say "send this to Chrome for implementation" is wrong. Chrome is a specialist consultant — deep frontend expertise, but zero project context.

The correct pattern: spawn team-agents within my session. They inherit my project context (conventions, constraints, deep-learn artifacts). A specialist oracle would have to re-learn the entire codebase first.

**spawn > delegate.** Always.

The corollary: when stuck on a domain question (responsive layout pattern, WCAG contrast rule, brand voice for Thai error messages), THEN consult the specialist via `maw hey`. Take their advice back and implement it myself (or through my team-agents). The specialist advises; PM executes.
