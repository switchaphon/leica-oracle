# Grill decisions must be saved to project memory AND source PRP immediately — not

Grill decisions must be saved to project memory AND source PRP immediately — not just in retro files. A 14-question grill session's decisions were lost because they only existed in a retrospective. Recovery cost 30 minutes + required a desktop chatlog.md. Prevention: save each decision inline as it's made, consolidate at end of grill.
