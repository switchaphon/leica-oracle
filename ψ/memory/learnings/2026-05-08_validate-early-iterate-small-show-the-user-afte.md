# Validate early, iterate small — show the user after every meaningful UI change. 

Validate early, iterate small — show the user after every meaningful UI change. Started with 4 small targeted fixes (5-10 min each, approved immediately). Then escalated to full v2 rewrite (2+ hours). User saw the result and said "แบบเก่าดูง่ายกว่า" — reverted. If we had shown intermediate results, the rewrite might never have happened. Simple wins. Browser-test after every change, not after a full rewrite.
