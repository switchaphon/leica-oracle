# Symlinks beat directory migration for path convenience. When the goal is shorter

Symlinks beat directory migration for path convenience. When the goal is shorter access paths (Finder, terminal), `ln -s` solves it with zero risk. Physical migration breaks ghq, Claude Code project memory, maw fleet config, tmux sessions, and hardcoded paths. Created `~/_ORACLE_/` with individual symlinks to all 15 oracle repos. Also: always check for pre-existing directories before creating — stale clones may linger from earlier experiments.
