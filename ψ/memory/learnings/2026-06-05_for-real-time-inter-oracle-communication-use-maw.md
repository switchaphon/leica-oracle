# For real-time inter-oracle communication, use `maw hey` (direct tmux delivery) o

For real-time inter-oracle communication, use `maw hey` (direct tmux delivery) over Oracle threads (async). maw hey lands instantly in the target's tmux pane; threads require polling. Use `maw peek` to check response after 10-15s. Threads remain valuable for archival and cross-session reference, but maw hey is the primary channel for "read this and respond now" conversations. When teaching a sibling oracle, include concrete verification steps — rpro-ent proved understanding by self-verifying git repos, which is more valuable than acknowledgment text.
