# Start minimal, add complexity only when simple proves insufficient. Built 5 iter

Start minimal, add complexity only when simple proves insufficient. Built 5 iterations of PrototypeGuide (beacons → callouts → step nav → auto-advance → triggers). Each layer created bugs. Final version: 80 lines, just beacon dots + bar. Rule: v1 = simplest thing that works. Add complexity only when users hit the wall of simplicity.
