# Commit Early + Short Sessions: commit prototype files after each functional mile

Commit Early + Short Sessions: commit prototype files after each functional milestone, don't accumulate hours of uncommitted work. Keep sessions to 2-3 hours with handover docs. One RRR per session at the end — not multiple checkpoints.
