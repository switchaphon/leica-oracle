# maw v26.5.2 Upgrade Patterns

1. **maw installs from GitHub, not npm** — `bun add -g maw-js@github:Soul-Brews-Studio/maw-js#v26.5.2`. The `bun update -g maw-js` pattern fails with 404 on npm registry.

2. **Core commands moved into bundled plugins** — v26.5.2's lean-core ships 22 commands without needing the online registry. Key renames: `workon` -> `wake`, `team` -> `swarm`, `capture` -> `peek`. `bud` is registry-only (use `/bud` skill instead).

3. **Claude Code `verbose: true` adds a built-in token counter** — redundant if you have a custom statusline showing context window usage. Set `verbose: false` in `~/.claude/settings.json` to remove it.
