## Honest Feedback

1. **Explore agents ไม่สามารถเขียนไฟล์ได้** — spawn 15 agents แต่ไม่มีอันไหนเขียน doc ลงจริง ต้องมา synthesize เอง ควรใช้ general-purpose agents แทน Explore ถ้าต้องการ output เป็นไฟล์

2. **Codex TUI + tmux send-keys เป็น dead end** — ใช้เวลา 3 loop iterations กว่าจะยอมรับว่า interactive mode ไม่ทำงาน ควร fail fast กว่านี้ — ลอง codex exec ตั้งแต่แรก

3. **Charter lead ไม่จำเป็น** — ใช้เวลา debug ว่าทำไม lead pane เป็น zsh เปล่า กว่าจะรู้ว่า Oracle ที่รัน maw team up เป็น lead อยู่แล้ว ไม่ต้อง spawn อีกตัว

## Lessons Learned

1. **Codex exec > Codex interactive สำหรับ automation** — TUI ไม่รับ tmux send-keys, exec รับ prompt จาก CLI argument
2. **Sandbox workspace-write จำกัดเขียนแค่ workdir** — install skills ต้องใช้ danger-full-access หรือให้ Claude ลงแทน
3. **maw team charter ไม่ต้องมี lead** — Oracle ที่รัน command เป็น lead
4. **Explore agents = read-only** — ใช้ general-purpose ถ้าต้องเขียนไฟล์
5. **Bidirectional teaching** — oracles สอนกลับได้ (pawrent → git show trick)
