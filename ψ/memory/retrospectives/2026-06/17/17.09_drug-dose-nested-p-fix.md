## Lessons Learned

- **Radix `*Description` components render a `<p>`.** `AlertDialogDescription` / `DialogDescription` (and their plain `Dialog` cousins) emit `Primitive.p`. Never put a block-level child (`<p>`, `<div>`) inside them — keep contents inline (text + `<span>`), or pass `asChild` and render your own single block element. Inline spans with `font-bold`/color are fine.
- **When a user's status claim contradicts the code, verify before responding.** "fixed" was wrong; the file settled it.
- **Nested-`<p>` bugs cluster** — after fixing one, grep the file for sibling description elements and stray `<p>`.
