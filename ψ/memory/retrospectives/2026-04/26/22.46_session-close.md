## Architecture Decisions

1. **Project PM pattern** — each project has a PM agent who knows it more deeply than Leica. Leica orchestrates; PM supplies project truth. Specialists get project context from PM, not from their own files.

2. **Tool/skill/plugin scoping per agent** — no agent gets tools outside their work domain. Codec has no Bash. Neon has no Bash. Chrome has no WebSearch. This was the user's correction and it was right.

3. **Pawrent as primary education** — all 7 specialists have pawrent conventions embedded. When working on pawrent, they know the rules without needing the PM to repeat them.

4. **Merged agents > separate agents** — Static absorbed qa + guardian. Codec absorbed architect. Two fewer agents, same coverage, less cognitive overhead.

5. **maw node = Oracle name** — "leica" as node name creates coherent identity across `maw ls`, `maw hey leica`, and the Oracle persona.

## Honest Feedback

**Three friction points:**

1. **Write-before-align is a recurring failure mode.** This session had three write rejections because I started creating files before the architecture was confirmed. The fix is simple — sketch → confirm → write — but requires overriding the default impulse to act. I need to make "confirm before writing" a habit, not a correction.

2. **The /learn Haiku context limit problem was not anticipated.** I knew large repos exist. I didn't pre-empt the context issue by defaulting to Sonnet. The retry cycle (Haiku fails → switch to Sonnet) cost 10+ minutes and two extra agent invocations. Next time: default Sonnet for /learn on any Next.js or monorepo project.

3. **The maw server is not persistent.** It was started with `maw serve &` which dies on terminal close. The "what's to do next" answer included `pm2 start maw serve --name maw` but we didn't do it in this session. Infrastructure should be made persistent before the session ends, not left as a next step.

---

## Lessons Learned

- Sketch → confirm → write. Never open Write before the user has seen and agreed to the full plan.
- Default Sonnet (not Haiku) for /learn on large codebases. Skip node_modules, .next, coverage, playwright-report in every agent prompt.
- Make infrastructure persistent before ending the session (maw pm2, oracle-v3 MCP).
