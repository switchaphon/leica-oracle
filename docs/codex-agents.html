Team Lifecycle (/codex-team)

Proven recipe from Nat's arra-oracle-v3 — claude lead + omx codex builders with branch isolation. Source: gist/nazt/319bce17

up — spawn # 1. Preflight check maw team preflight ψ/teams/[project]-team.yaml # 2. Spawn (creates tmux windows + worktrees) maw team up [project]-team # 3. Verify for N in 1 2 3; do maw peek [session]:codex-$N 2>&1 | tail -5 done
dispatch — assign task # Use maw hey (NOT tmux send-keys) maw hey codex-1 "You are Chrome. Read agents/chrome.md. Task: build AppointmentPicker"
status — check all for N in 1 2 3; do echo "=== codex-$N ===" maw peek [session]:codex-$N 2>&1 | tail -10 done # Check open PRs gh pr list --base alpha --state open
down — teardown # Auto-saves WIP, kills windows, removes worktrees, keeps branches maw team down [project]-team
restart / relaunch # Full restart maw team down [project]-team && sleep 3 && maw team up [project]-team # Single dead member only maw team up [project]-team --only codex-3
single agent (no team) # Quick fire-and-forget codex exec -s workspace-write "Read AGENTS.md. Task: refactor component X"

Key Rules (from Nat's 7 rules)
📄
Charter is source of truth
ψ/teams/*.yaml (brain-committed) — NOT .maw/teams/ or ~/.claude/teams/
📨
Dispatch via maw hey only
Never maw team send or SendMessage for omx coders — silent no-op on omx
🔍
Confirm before relaunch
maw peek to verify dead — "may not have submitted" is a false negative
🚫
Never push to main / never auto-merge
All work targets alpha. Oracle reviews PRs — human approves merges
When to Promote to Codex Oracle

AGENTS.md is sufficient now. Promote to a full Codex Oracle only when these triggers appear:

🔄
Repeated mistakes
Codex makes the same error across sessions — needs retrospective memory
🔗
Codex-to-Codex coordination
Multiple Codex agents need to integrate work — needs a coordinator with state
📋
Standing orders that change
Dynamic rules that evolve faster than AGENTS.md updates — needs living memory
🌙
Autonomous routines
Nightly builds, auto-fix, batch migration — Codex needs to self-manage
